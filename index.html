Planning tools that understand code

Not just a pretty chart. RoadMap understands your git history, issues, and PRs.

📊

Gantt Charts

Interactive timeline visualization with drag-to-resize tasks, dependency arrows, critical path highlighting, and milestone markers. Zoom from day view to year view with a scroll.

📄

Kanban Boards

Drag-and-drop cards across customizable columns. WIP limits, swimlanes by assignee or repo, auto-archive completed tasks. Syncs bidirectionally with Gitea and GitHub issues.

🔗

Dependency Tracking

Define task dependencies with finish-to-start, start-to-start, and finish-to-finish relationships. Automatic critical path calculation. Cascade delay warnings when blockers slip.

🧠

AI Timeline Estimation

Analyze your historical commit velocity, PR merge times, and issue resolution patterns to predict realistic timelines. No more "this will take two weeks" turning into two months.

🔭

Multi-Repo Views

See work across all your repositories in a single unified view. Track a feature that spans the frontend, backend, and infrastructure repos as one cohesive project.

📈

Velocity Metrics

Track throughput, cycle time, lead time, and burndown across sprints. Historical comparison shows if your team is accelerating or hitting friction. Per-assignee and per-repo breakdowns.

roadmap.config.ts 
// RoadMap — visual project planning with git integration
import { RoadMap } from '@blackroad/roadmap';

const project = new RoadMap({
  name: 'BlackRoad OS v3.0',
  repos: [
    { url: 'gitea://octavia/blackroad-operator', label: 'Core' },
    { url: 'gitea://octavia/blackroad-web',      label: 'Frontend' },
    { url: 'gitea://octavia/roadblock',          label: 'Security' },
  ],
  views: ['gantt', 'kanban', 'timeline'],
  ai: {
    estimation: true,           // AI-powered timeline prediction
    model: 'mistral:7b',         // local inference for estimates
  },
  sync: {
    gitea: true,                // 2-way issue sync with Gitea
    github: true,               // 2-way issue sync with GitHub
  }
});

// Auto-imports all open issues from connected repos
await project.sync();
// AI estimates: "Core engine: 14 weeks (based on team velocity)"

# Install RoadMap

# Or with Homebrew
brew install blackroad/tap/roadmap

# Initialize in your project
roadmap init --repos ./my-app ./my-api

# Connect to Gitea
roadmap connect gitea --url octavia.local:3100

# Open the dashboard
roadmap serve --port 4000
